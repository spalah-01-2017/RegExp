Условия

  1. Количество слов в имени: 2-3 (можно заменить это на условие 1-3 слова в имени)
  2. Каждое слово начинается с большой буквы, за которой идут маленькие
  3. Минимальная длина слова - 2 символа
  4. В имени не должно быть никаких символов, кроме a-z
  5. Между словами имени должен быть один пробельный символ
    1. Наличие поля ввода (input)
    2. Срабатывание проверки на правильность ввода при каждом нажатии клавиши пользователем
    3. Отображение результата проверки рядом с полем ввода (valid/invalid), либо другое визуальное оповещение для пользователя, не используя стандартные браузерные функции (alert)
